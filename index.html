Features

Prison is built to run a prison server of any size with speed and elegance. It has a modular design, which makes it a powerful and extensible platform for you to create your server on.

Mines

Prison comes with an incredible mines module that is designed to be powerful and lag-free. It boasts asynchronous generation, signs, multiple reset modes, and more features that could previously only be found in premium plugins. For free.

Ranks

If you thought mines were good, wait till you see ranks. Prison's ranks module not only allows users to rank up, but also hooks in with your permissions plugin to give you total control over when and where users get certain permissions.

Shops

Shops are important on prison servers, and Prison doesn't skimp on these. With features such as bulk selling, contextual shops, and more, your players will be selling stuff faster than Pearson sells books.

Extensions

Extensions will add even more features to your prison server for a low price. They hook right into Prison and deliver the same amount of performance as you'd expect from Prison. They will hopefully be released near the end of Q3.
