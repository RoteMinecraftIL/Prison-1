Prison 3.0 is here.

Totally rethinked, Prison 3 will give your server everything it needs
to run a professional, efficient, and feature-packed server.

  • Mines

    Features such as composition preloading and multiple reset methods ensure that your server has diverse mines with little performance impact. Signs and percentage-based resets are some of the many included features.

  • Ranks

    Prison provides an incredibly customizable ranks system. It includes the big things - rankups using XP or money, kits, warps, build restriction, permissions - as well as the small things, such as dual prefixes and commands that are run upon a rankup.

  • Shops

    Shops are a big part of prison servers, so why settle for the average shop? The shops system provides contextual shops, which adjust based on the user's rank and mine that they're in, as well as quick selling, inventory shops, and signs.

  • Extensions

    Extensions are paid plugins that add on to the features of Prison. They will be released sometime in the future, starting at $5.00 apiece. Planned extensions include cells, guards, prestige ranks, and tree mines.
