Installation

Estimated time: 5 minutes

So, you've decided to install Prison! You are one step closer to creating the best prison server on the planet. Before you dive into the awesomeness, you're going to have to install Prison on your server. Through some short and easy steps, you'll be up and running in no time.

  1. Stop your server.
  2. Check that you have one of the following permissions plugins installed: GroupManager, PermissionsEx, or any other permissions plugin supported by Vault (provided you have Vault installed).
  3. Check that you have one of the following economy plugins installed: Essentials, iConomy, or any other economy plugin supported by Vault (provided you have Vault installed).
  4. Check that you have WorldEdit installed.
  5. Download the latest version of Prison.
  6. Drag Prison.jar into your plugins folder.
  7. Start your server.
That was easy! Type /prison status to ensure that everything's working correctly.
